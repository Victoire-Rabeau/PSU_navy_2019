# navy

## Description

The aim of this project is to implement a local multiplyer battleship game on a terminal.

## Getting started

Compile with
```bash
make
```

## Usage    

player 1:
```bash
./navy [navy_positions]
```
player 2:
```bash
./navy [first_player_pid] [navy_positions]
```
